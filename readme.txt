Пояснение названий полей:

today                   Сегодняшняя дата
callsign                Кодовое название судна
flag                    Под чьим флагом идет
mmsi                    Код судна
current_position        Позиция судна на момент pos_recieved
cord_n                  Координата N
cord_w                  Координата W
pos_recieved            Когда получена позиция судна
speed                   Скорость судна в узлах

status{                 Статус судна
    moored              Пришвартовано
    under way           В пути
    at anchor           На якоре
}                  
destination             Пункт назначения
ETA                     Ожидаемые дата и время прибытия
reference               Ссылка на карту с расположением судна
